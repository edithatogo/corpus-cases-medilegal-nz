# Track Specification: Integrate MoJ Tribunals
## 1. Goal
Add Ministry of Justice (MoJ) Tribunals decisions to the NZ medical-legal corpus.
## 2. Source
URL: https://www.justice.govt.nz/tribunals/
## 3. Data Format
Follows the standard format (Markdown+YAML, Text+JSON, JSONL, Parquet).
## 4. Pipeline Integration
Configured through `moj_tribunals_pipeline.yaml` and the multi-source pipeline framework.
