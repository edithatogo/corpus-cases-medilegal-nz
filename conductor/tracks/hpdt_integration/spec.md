# Track Specification: Integrate HPDT
## 1. Goal
Add Health Practitioners Disciplinary Tribunal (HPDT) decisions to the NZ medical-legal corpus.
## 2. Source
URL: https://www.hpdt.org.nz/Search-Decisions
## 3. Data Format
Follows the standard format (Markdown+YAML, Text+JSON, JSONL, Parquet).
## 4. Pipeline Integration
Configured through `hpdt_pipeline.yaml` and the multi-source pipeline framework.
